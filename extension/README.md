# 打ち合わせ候補日時生成 Chrome拡張機能

Googleカレンダーの予定を読み取り、空き時間から打ち合わせ候補日時を生成し、指定フォーマットでクリップボードにコピーするChrome拡張機能です。

## 機能

- Googleカレンダーから予定を自動取得
- 空き時間を自動計算
- 指定した条件に基づいて候補日時を生成
- ワンクリックで候補をクリップボードにコピー
- 設定を保存して次回も利用可能

## インストール方法

### 1. Google Cloud Consoleでの設定

拡張機能を使用する前に、Google Calendar APIを有効にし、OAuth 2.0クライアントIDを取得する必要があります。

1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. 新しいプロジェクトを作成
3. 「APIとサービス」→「ライブラリ」から「Google Calendar API」を検索して有効化
4. 「認証情報」→「認証情報を作成」→「OAuthクライアントID」を選択
5. アプリケーションの種類として「Chromeアプリ」を選択
6. クライアントIDをコピー

### 2. manifest.jsonの編集

`extension/manifest.json`を開き、以下の行を編集します：

```json
"client_id": "YOUR_CLIENT_ID.apps.googleusercontent.com"
```

取得したクライアントIDに置き換えてください。

### 3. Chromeへの読み込み

1. Chromeブラウザで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `extension`フォルダを選択

### 4. アイコンの準備（オプション）

`extension/icons/`ディレクトリに以下のアイコンファイルを配置してください：
- icon16.png (16x16ピクセル)
- icon48.png (48x48ピクセル)
- icon128.png (128x128ピクセル)

アイコンがない場合は、開発中は動作しますが、警告が表示されます。

## 使い方

1. Chromeのツールバーにある拡張機能アイコンをクリック
2. 必要に応じて設定を調整：
   - **候補数**: 生成する候補の数（1〜10）
   - **会議時間**: 会議の予定時間（30分/60分/90分）
   - **抽出開始時間**: 候補を探す時間帯の開始時刻
   - **抽出終了時間**: 候補を探す時間帯の終了時刻
   - **曜日**: 候補を探す曜日（複数選択可）
   - **検索期間**: 何日先まで検索するか（1〜30日）
3. 「候補を生成」ボタンをクリック
4. 初回は Googleアカウントへのアクセス許可を求められます
5. 候補が自動的にクリップボードにコピーされます
6. メールやSlackなどに貼り付けて使用

## 出力フォーマット

候補は以下の形式で出力されます：

```
3月15日（金）10時00分〜11時00分
3月15日（金）13時00分〜14時00分
3月16日（土）11時00分〜12時00分
3月17日（日）14時00分〜15時00分
3月18日（月）10時00分〜11時00分
```

## 設定の保存

設定は自動的にブラウザに保存され、次回使用時にも維持されます。

## トラブルシューティング

### 「認証エラー」が表示される

1. manifest.jsonのclient_IDが正しく設定されているか確認
2. Google Cloud ConsoleでGoogle Calendar APIが有効になっているか確認
3. 拡張機能を一度削除して再読み込み

### 「候補が見つかりませんでした」と表示される

1. 検索期間を延ばしてみる
2. 時間帯を広げてみる（例：9:00-20:00）
3. 曜日の選択を増やす

### カレンダーが取得できない

1. Googleカレンダーにログインしているか確認
2. カレンダーに予定が入っているか確認
3. ブラウザのコンソールでエラーメッセージを確認

## ファイル構成

```
extension/
├── manifest.json          # 拡張機能の設定ファイル
├── popup.html            # ポップアップのUI
├── popup.js              # UIのロジックと設定管理
├── calendar.js           # Google Calendar API連携
├── slot-generator.js     # 空き時間計算ロジック
├── formatter.js          # 出力フォーマット処理
├── background.js         # バックグラウンドスクリプト
├── style.css             # スタイルシート
└── icons/                # アイコンファイル
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## 開発

### 前提条件

- Google Chrome
- Googleアカウント
- テキストエディタ

### デバッグ

1. `chrome://extensions/` で拡張機能の詳細を開く
2. 「バックグラウンドページを検証」をクリック
3. ポップアップを右クリック→「検証」でDevToolsを開く

### ローカルテスト

1. コードを編集
2. `chrome://extensions/` で「更新」ボタンをクリック
3. 拡張機能のアイコンをクリックしてテスト

## 制限事項

- Googleカレンダーのメインカレンダーのみ対応
- 共有カレンダーには非対応
- 終日イベントは考慮されません
- 最大30日先までの検索
